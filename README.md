# Vamsee AI — Personal PC Assistant

An autonomous AI Agent that runs locally on your computer to assist with daily tasks:
✔ **Control your PC** (Apps, menus, inputs)
✔ **Screen AI** (Real-time reading and fixing errors)
✔ **File Manager** (Auto-organising Downloads)
✔ **Task Management** (Dashboard summaries & stats)
✔ **Voice Assistant** (State-machine listening & speech)

---

## Project Structure
- **/frontend**: Next.js desktop UI framework with dark purple premium aesthetics
- **/backend**: AI processing & OS Automation logic

## Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
